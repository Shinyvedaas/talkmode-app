# TalkMode

Public community home for **TalkMode** — a real-time Korean voice assistant for macOS.

- **Website & download** — https://talkmode.baryon.ai
- **Issues** — bug reports and feature requests in the [Issues](https://github.com/baryonlabs/talkmode-app/issues) tab
- **Discussions** — questions, ideas, and help in the [Discussions](https://github.com/baryonlabs/talkmode-app/discussions) tab

The application source is maintained in a separate repository; this repository
hosts community issues and discussion. TalkMode is free software under the
GNU Affero General Public License v3.0.
